# PortusSophia™ Governance Repository

## Repository Scope

This repository contains **governance-only** materials for the PortusSophia™ project. It defines and enforces structural standards, metadata rules, naming conventions, and stewardship workflows.

## What Belongs Here

- Universal Informational Content Header (UICH) templates and specifications
- Metadata schemas and validation rules
- Path and naming conventions
- Structural authorities and standards
- Stewardship workflows and processes
- Compliance artifacts and checklists
- Trademark usage guidelines

## What Does NOT Belong Here

This repository explicitly **excludes**:

- ❌ Canon content or narrative material
- ❌ Public-facing promotional content
- ❌ WebKernel implementation code
- ❌ Story elements or worldbuilding
- ❌ Character profiles or lore
- ❌ User-facing documentation (unless governance-related)

## Redirect Policy

If requests are made for canon, narrative, or public-facing content:

1. **Stop** — Do not create such content here
2. **Warn** — Clarify this is a governance-only repository
3. **Redirect** — Point to the correct repository:
   - Canon/narrative → appropriate narrative repository
   - Public-facing → marketing/documentation repository
   - WebKernel → implementation repository

## Governance Discipline

All contributions must:

- Maintain **neutral, technical tone**
- Follow established standards without invention
- Preserve all **PortusSophia™** trademark forms
- Respect sealed artifacts (read-only governance documents)
- Enforce structure without modifying core standards

## Trademark Usage

Always use the proper trademark format: **PortusSophia™**

Never use:
- Portus Sophia
- PortusSophia (without ™)
- portussophia
- Other variations

## Questions?

For governance-related questions, open an issue in this repository.

For non-governance questions, please direct inquiries to the appropriate repository based on the topic.
